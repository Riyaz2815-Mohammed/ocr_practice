# STATIC WEB PAGE USING HTML.

Arm:-

To create a static webpage with headings, paragraphs, images and links that demonstrate basic HTML elements and section navigation procedure:-

Create a project folder and a new HTML file (Index.html)

Set up the basic html structure with Lhtml, Lhead, and Lbody.

Add headings, paragraphs and sections like Home, Eligibility and Facility.

Invect navigation links to move between sections on the same page

Add images and notes then if required the horizontal lines or links to organize

Save the file and open it in a browser to check content and layout

Make corrections or enhancements as needed.
